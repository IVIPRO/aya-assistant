Implement an animated teacher for the Junior / Начален етап module.

Feature name:
AYA Panda Animated Teacher

Goal:
Add a friendly animated teaching companion that appears across the Junior learning experience and reacts to learning events.

Create:
- a reusable AnimatedTeacher component
- localized speech bubble messages
- simple animation states
- floating helper UI without breaking existing layout

Technical requirements:

1. Create component:
src/components/AnimatedTeacher.tsx

2. Use lightweight animation only:
- CSS animations and/or Framer Motion
- must work well on desktop and mobile
- keep performance light

3. Supported animation states:
- idle
- talking
- happy
- thinking
- encouraging

Behavior:
- idle: gentle breathing + blinking
- talking: slight bounce while speech bubble is visible
- happy: tiny jump + sparkle effect
- thinking: subtle tilt / thinking pose
- encouraging: gentle nod

4. Character:
Use AYA Panda as the default animated teacher for Junior / Начален етап.

5. Placement:
Show AnimatedTeacher in:
- Junior dashboard / welcome card
- subject lesson pages
- mission pages
- Ask AYA chat screen
- Homework Vision support area

Default position:
- bottom right floating assistant
- should not block main buttons or inputs
- responsive on mobile and desktop

6. Speech bubble:
Add a speech bubble above the teacher.

7. State triggers:
AnimatedTeacher state should react to learning events:

- on page load -> idle
- when a lesson opens -> talking
- when the child gets a correct answer -> happy
- when the child gets a wrong answer -> encouraging
- when waiting for child input -> thinking
