====================================
3. Dynamic Task Generation Rules

Create a local math task generator.

A task object should include:
- id
- expression
- answer
- missionType
- difficulty
- createdAt

Example:
{
  id: "task_001",
  expression: "3 + 4",
  answer: 7,
  missionType: "addition_to_10",
  difficulty: "easy"
}

Mission 1: Събиране до 10
Rules:
- a = random integer 0..10
- b = random integer 0..10
- a + b <= 10

Mission 2: Изваждане до 10
Rules:
- a = random integer 1..10
- b = random integer 0..a
- result >= 0

Mission 3: Събиране до 20
Rules:
- a = random integer 0..20
- b = random integer 0..20
- a + b <= 20

Prevent repeating the last 5 generated tasks within the same mission session.

====================================
4. Mission Flow

When a child starts a mission:

1. Generate the first task
2. Show it in the mission/chat area
3. Let the child enter an answer
4. Check correctness
5. If correct:
   - mark task solved
   - award +5 XP
   - generate next task
6. When mission target is reached:
   - show mission complete message
   - award bonus XP

Mission targets:
- Събиране до 10 -> 5 correct tasks
- Изваждане до 10 -> 5 correct tasks
- Събиране до 20 -> 6 correct tasks

====================================

AYA Panda should present tasks in a child-friendly teacher tone.
