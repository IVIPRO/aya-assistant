STEP 1 — FULL FINAL AUDIT
==================================================

Inspect the current codebase and verify the app is hardened for MVP production use.

Audit all of these areas:

1. Runtime stability
- no crash-prone critical flows
- no unsafe null/undefined access in main user paths
- safe guards around optional data
- safe loading / empty / fallback states

2. Data safety
- no duplicate writes in critical learning flows
- progress writes are idempotent enough
- XP / badges / streak / mastery do not corrupt state
- parent analytics reads only real persisted data
- no child-to-child leakage

3. Auth / access safety
- protected routes behave safely
- parent vs child boundaries remain enforced
- logout / refresh do not expose wrong data
- child switching remains isolated

4. Curriculum safety
- grade restrictions remain enforced
- forbidden topics do not leak
- junior and прогимназия stay separated correctly
- generated tasks remain grade-appropriate

5. Navigation safety
- no dead ends in core flow
- no broken back navigation in major screens
- no redirect loops
- no lesson/practice/session loops

6. Voice / media safety
- voice toggle safe
- no autoplay chaos
- no overlapping playback
- safe behavior if speech unavailable or disabled

7. Persistence safety
- refresh-safe where intended
- no broken state restore
- no invalid cached UI
- no duplicate milestone triggering after reload

8. Parent dashboard safety
- real metrics only
- safe fallbacks if partial data missing
- recommendations do not break page
- weekly/daily/report data safe

9. Junior / child UX safety
- lessons remain usable
- wrong-answer explanations safe
- adaptive difficulty safe
- streak / celebration / gamification does not spam or break flow

10. Production readiness hygiene
- no obvious debug remnants in critical user flows
- no placeholder blocker text in main screens
- no fake data in production-critical screens
- TypeScript stays clean
- existing architecture remains stable
