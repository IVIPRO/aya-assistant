INTEGRATION FLOW

Replace direct task generation with the validated flow.

Use the pattern:

const validated = validateAndCorrectContext(userContext)
const correctedContext = validated.correctedContext || validated.context
const task = generateTask(correctedContext)

This ensures:
- grade rules respected
- topics valid
- safe fallback behavior

log it internally but keep UI clean.
