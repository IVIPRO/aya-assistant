--------------------------------
WHAT TO BUILD
--------------------------------

Create a lightweight rule-based wrong-answer diagnosis layer.

Add a helper such as:

diagnoseWrongAnswer(params)

Input should include:
- grade
- topic
- task type
- question text
- correct answer
- user answer

Output should include:
- errorType
- shortBulgarianExplanation
- optionalHint

--------------------------------

For Grade 2 Math:
- wrong counting
- forgot carrying / regrouping
- digit reversal
- chose subtraction instead of addition
- chose addition instead of subtraction
- misunderstood word problem
- off by 1
- off by 10

For Grade 5 Math:
- decimal place misalignment
- ignored decimal comma/point logic
- wrong operation chosen
- algebra sign error
- forgot to move term across equal sign correctly
- fraction numerator/denominator confusion
- incorrect comparison of decimal numbers
- word problem operation misunderstanding
- arithmetic slip
