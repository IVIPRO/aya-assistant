- 1 клас

Before adding content, determine whether the current architecture expects:
- topic-based lessons

- skills arrays
- difficulty levels
- localized titles

Reuse the existing exact structure.

--------------------------------------------------
STEP 4 — IMPLEMENT ONLY MISSING CURRICULUM CONTENT

Add missing lessons only where gaps are confirmed.

But use only where grade-appropriate and only where missing.
