Create a minimal internal AYA Lesson Generator dev utility that can generate lesson seed content using the existing architecture.

- curriculum.ts

Step 1 — Audit existing generators
Check whether mathTaskGenerator.ts already provides reusable helpers for generating math problems or quizzes.

reuse its logic instead of creating a second generator.

Step 2 — Create minimal generator utility
Create ONE small dev-only generator utility file:

api-server/src/tools/lessonGenerator.ts

This file must generate lesson seed content compatible with the EXISTING schema used in lessonContent.ts.

Input parameters:
- grade
- subject
- topic
- lessonCount

Example:

generateLessons({
  grade: 2,
  subject: "mathematics",
  topic: "multiplication_intro",
  lessonCount: 5
})

- lesson
- practice
- quiz

Do not invent a new schema.
